# 分时图逐笔数据指标实现计划

## 功能需求
在分时图上方添加一个新的视图，包含：
1. **今天的逐笔交易走势图**（价格走势图）
2. **技术指标图**：累计买入/卖出线、买入/卖出加速度指标
3. **大/中/小单资金流入流出统计柱状图**

## 实现步骤

### 1. 添加逐笔数据API接口 (stockApi.ts)
- 新增 `getTodayTradeData` 函数：获取今天的全天逐笔成交数据
- 新增接口定义：`TradeData` (包含 Price, Time, Volume, Status)

### 2. 新增数据类型定义 (types/stock.ts)
- 定义 `TradeTick` 接口：逐笔成交数据结构
- 定义 `TradeIndicatorStats` 接口：逐笔指标统计数据
- 定义 `CapitalFlowStats` 接口：资金流入流出统计数据

### 3. 新增逐笔数据处理工具函数 (utils/tradeData.ts)
- `convertTradeToChart`：将逐笔数据转换为图表数据
- `calculateCumulativeBuySell`：计算累计买入/卖出线
- `calculateBuySellAcceleration`：计算买入/卖出加速度
- `calculateCapitalFlow`：计算大/中/小单资金流入流出统计

### 4. 创建逐笔走势图组件 (components/TradeTrendChart.tsx)
- 使用 Canvas 绘制逐笔价格走势图
- 支持鼠标悬停显示详细信息
- 支持缩放和拖拽

### 5. 创建技术指标图组件 (components/TradeIndicatorChart.tsx)
- 绘制累计买入/卖出线
- 绘制买入/卖出加速度柱状图
- 支持图例说明

### 6. 创建资金流入流出统计组件 (components/CapitalFlowChart.tsx)
- 绘制大/中/小单资金流入流出柱状图
- 支持颜色区分（流入绿色，流出红色）
- 显示金额统计

### 7. 创建主容器组件 (components/TradeAnalysisPanel.tsx)
- 整合上述三个组件
- 管理数据加载状态
- 提供时间范围选择（今天）

### 8. 集成到主应用 (App.tsx)
- 在分时图模式下显示逐笔分析面板
- 在非分时图模式下隐藏或禁用

## 数据结构设计

### TradeTick (逐笔成交数据)
```typescript
interface TradeTick {
  time: string;           // 时间 "09:31"
  price: number;          // 价格
  volume: number;         // 成交量
  status: 0 | 1 | 2;     // 0=中性, 1=买入, 2=卖出
  amount: number;         // 成交额
}
```

### TradeIndicatorStats (逐笔指标统计)
```typescript
interface TradeIndicatorStats {
  cumulativeBuy: number[];    // 累计买入
  cumulativeSell: number[];   // 累计卖出
  buyAcceleration: number[];  // 买入加速度
  sellAcceleration: number[]; // 卖出加速度
}
```

### CapitalFlowStats (资金流入流出统计)
```typescript
interface CapitalFlowStats {
  largeInflow: number;    // 大单流入
  largeOutflow: number;   // 大单流出
  mediumInflow: number;   // 中单流入
  mediumOutflow: number;  // 中单流出
  smallInflow: number;    // 小单流入
  smallOutflow: number;   // 小单流出
}
```

## 订单分类规则
- **大单**：单笔成交额 >= 100万
- **中单**：单笔成交额 20万-100万
- **小单**：单笔成交额 < 20万

## 注意事项
- 逐笔数据量可能较大（今天约4000+条），需要优化渲染性能
- 使用 Canvas 而非 SVG 来提高渲染性能
- 数据加载时显示 loading 状态
- 错误处理：如果非交易日或无数据，显示提示信息
